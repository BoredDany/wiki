Herramientas de deployment

Docker

Docker nos permite contenerizar nuestra aplicación y sus dependencias, facilitando el despliegue consistente en cualquier entorno.

Jenkins

Jenkins automatiza las fases de prueba y despliegue, permitiéndonos integrar cambios de forma continua.

Docker

Kubernetes orquesta nuestros contenedores Docker, gestionando el escalado automático y la distribución de carga para manejar eficientemente los recursos y mantener la aplicación en funcionamiento de manera continua.
